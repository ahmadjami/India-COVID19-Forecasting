# GOAL
Build a symbol-agnostic regression model that predicts forward short-horizon
returns, for execution signal generation. THREE separate models, one per horizon:
5 min, 30 min, 60 min.

# DESIGN DECISIONS ALREADY MADE (follow these)
1. Bars: start with VOLUME bars (conflation preserves size, so volume bars are
   robust to the tick/conflation differences). Dollar bars later as an upgrade.
2. Bar threshold: PER-SYMBOL, from a TRAILING (causal) volume estimate, targeting
   ~N bars/session. Never full-sample (leakage).
3. Bars must NOT cross session boundaries — reset accumulator at lunch, close,
   and day boundaries. Build bars per (symbol, date, session) group.
4. Drop incomplete final bars (partial volume scale).
5. Symbol-agnostic = symbol-agnostic MODEL but symbol-AWARE normalization:
   - No symbol/market/source_file as features.
   - Features dimensionless (OBI, relative spread in bps, microprice offset)
     or self-standardized causally per symbol (trailing window).
6. Features computed from data up to time t only. Same feature matrix X for all
   three horizons; only the label changes.
7. Target: forward MICROPRICE return (microprice = (bid*asize + ask*bsize)/
   (bsize+asize)), in bps, then normalized by trailing realized vol matched to
   each horizon ("sigma units"). Predicting CHANGE, not level.
8. Per-horizon purge + embargo in walk-forward CV, sized to the horizon
   (>=5/30/60 min). Long-horizon labels overlap heavily — consider uniqueness/
   sample weighting.
9. Model: CatBoost regressor, pooled across all symbols/markets.
10. Evaluate with rank/sign quality (information coefficient / directional
    accuracy), not just RMSE.
